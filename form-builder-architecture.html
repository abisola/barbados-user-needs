GovTech Barbados Form Prototype Generator – Platform Architecture

Six-layer stack powering form prototype generation, submission, and conversational form-filling for the Government of Barbados. Lower layers provide shared infrastructure – upper layers deliver citizen and designer experiences.

Key Data Flows
🔄
Prototype Generation
PDF spec → pdf-parse → Claude Opus stream → auto-continue if truncated (up to 5 calls) → stitch HTML → S3 upload → URL returned
📨
Form Submission
GovBB.next() → POST /api/submit → generate ref (PREFIX-YEAR-XXXX) → parallel SES emails → ref on confirmation page
💬
Chat Form-Filling
chat.html extracts form script → Claude Sonnet multi-turn → tool loop (save_fields, lookups, complete_form) → triggers /api/submit
🪪
Citizen Lookup
GovBB.tridentIdLookup() → POST /api/trident-id → citizen data → confirm → populates name, DOB, address, etc.
🚗
Vehicle Lookup
GovBB.vehicleLookup() → POST /api/vehicle-lookup → make, model, colour, engine/chassis → confirm → populates fields
🏢
Business Lookup
GovBB.businessLookup() → POST /api/business-lookup → CAIPO entity data, TIN, NIS, directors → confirm → populates fields
